# money-printing-impact-analysis
Data analysis project using real-world macroeconomic data to study the impact of money supply growth on inflation and currency value.

# Impact of Money Printing on Inflation and Currency Stability

## Objective
Analyze how changes in money supply affect inflation, exchange rates, and GDP using real-world macroeconomic data.

## Data Sources
- World Bank
- IMF
- Federal Reserve

## Tools Used
- SQL
- Excel
- Power BI
